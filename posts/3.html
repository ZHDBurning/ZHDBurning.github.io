段落

要创建段落，分隔一行即可

我是一个段落

我是真的很喜欢使用Markdown

换行符

使用<br>,在一个段落里产生换行。

这是第一行。
这是第二行

强调（加粗）

您可以通过将文本设置为粗体来增加重点，使用** **或者__ __ ，注意下划线的开头和结束要空格，不然没有效果，这两种方式有细微的样式差异，第二种带点空隙。

我只喜欢粗体字

arry老师 我爱你 一天 _下划线不推荐使用

斜体

要使文本变为斜体，请在单词或短语前后添加一个星号或下划线，
同样第二种方式前后需要空格，才能生效，由于兼容性，我们最好使用第一种带星号的方式

我也特别喜欢斜体字

是吗，真的喜欢 斜体 字吗？_下划线不推荐使用

引用

要创建块引用，请将 > 在段落前添加一个。

我是一个块引用

带有多个段落的块引用

块引用可以包含多个段落。将 > 在段落之间的空白行上添加一个

渲染的输出如下所示：

我是陈冠希

全宇宙最帅气的男神

嵌套块引用

块引用可以嵌套。将 >> 在要嵌套的段落前面添加一个

我是陈冠希

全宇宙最帅气的男神

你最棒

有序列表

创建有序列表

  1. 第一项
  2. 第二项
    1. 缩进的项目一
    2. 缩进的项目二
  3. 第三项
  4. 第四项
  5. 第五项

无序列表

要创建无序列表，请在项目前添加破折号（-）、星号（*）或加号（+），
最好使用破折号

  • 第一项
  • 第二项
    • 缩进项目一
    • 缩进项目二
      • 缩进缩进项目1
      • 缩进缩进项目2
    • 缩进项目三
  • 第三项
  • 第四项

块引用

  • 我是第一个列表项
  • 我是第二个列表项

    快引用在第二个列表项下方看起来很棒

  • 我是第三列表项

插入图片

插入图片的两种方式，可以在img标签里设置样式，调整图片大小

代码

要将单词或短语表示为代码，请将其括在反引号（ ` ）中

语义元素示例：<from>、<table>、<nav>和<header>有明确定义其内容

转义反引号

如果要表示为代码的单词或短语包含一个或多个反引号，则可以通过将单词或短语括在双反引号（``）中来对其进行转义。

Use `code` in your Markdown file.

代码块

<html>
    <head></head>
</html>

水平线

要创建水平线，请在一行上单独使用三个或更多星号（***）、破折号（—）或下划线（___），推荐使用破折号（—）。

链接

要创建链接，请将链接文本括在括号中（例如，[Duck Duck Go]），然后紧跟在括号中的URL

渲染的输出如下所示：

我最喜欢的技术博客是：arry老师的博客

我最喜欢的技术博客是：**arry老师的博客**

我最喜欢的技术博客是：arry老师的博客

渲染的输出如下所示：

https://arryblog.com

arry@163.com

图片

要添加图像，请添加感叹号（！），后跟括号中的替代文本，以及括号中的图像资源的路径或URL，您可以选择在路径或URL后添加引号中的标题。

转义字符

要显示原本用于在 Markdown 文档中格式化文本的文字字符，请在字符前面添加反斜杠（\）。

渲染的输出如下所示：

* 如果没有反斜杠，这将是列表项目的符号

- 如果没有反斜杠，这将是无序列表

# 如果没有反斜杠，这将是标题一

HTML 最佳实际

您不能在块级 HTML 标记中使用 Markdown 语法，例如，<p>italic and **bold**</p>不会生效

渲染的输出如下所示：

italic and **bold**

表格

要添加表格，请使用三个或更多连字符（—）来创建每一列的标题，并使用竖线（|）分隔每一列。为了兼容性，您还应该在行的任一端添加一个管道

渲染的输出如下所示：

语法 描述
标题 标题
段落 段落

表格内容的对齐方式

您可以通过在标题行内连字符的左侧、右侧或两侧添加冒号（：）来将列中的文本向左、向右或居中对齐。

渲染的输出如下所示：

语法 描述 测试文本
标题 标题 测试1
段落 段落 测试2

格式化表格中的文本

您可以格式化表格中的文本。例如，您可以添加链接、代码（仅在反引号（`）中的单词或短语，而不是代码块）和强调。

语法 描述 测试文本
标题 标题 测试1
段落 段落 测试2

围栏代码块

基本的Markdown语法允许您通过将行缩进四个空格或一个制表符来创建代码块，
如果你觉得不方便，请尝试使用围栏代码块。根据您的Markdown处理器或编辑器，您将在代码块前后的行上使用三个反引号（```）

标题 ID

许多 Markdown 处理器支持标题的自定义ID——一些 Markdown 处理器会自动添加它们，添加自定义 ID 允许您直接链接到标题并使用 CSS 修改它们。要添加自定义标题 ID，请将自定义 ID 括在与标题相同的行上的花括号中。
